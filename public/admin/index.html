Dreamina Account Pool

把账号池、Session、密钥和调度状态压进同一个运维控制台。

这个后台面向 Dreamina 号池运营场景，重点服务账号可用性维护、 API Key 发放、Session 续期和健康调度，不再让关键状态散落在脚本、 Docker 日志和临时表格里。

Session 自动续期 黑名单可追溯 并发租约透明 OpenAI 兼容出口

Ops Focus

进入后台后先看运行状态，再处理账号与密钥。

账号侧 健康 / 黑名单 / Session

把是否能接单、是否该续期直接暴露在一个视图里。

调用侧 Key 能力 / 最近使用 / 签发结果

避免外部调用方状态不透明，创建后可以立刻复制交付。

安全侧 管理员会话 / 密码更新 / 最近动作

敏感操作集中在右侧安全区，减少误触和视线切换。

Admin Login
